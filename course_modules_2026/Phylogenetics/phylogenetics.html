1. Phylogenetic Tree Inference

1.1 Tree Topology

A phylogenetic tree represents evolutionary relationships among sequences. The topology describes how taxa are connected, independent of branch lengths.

1.2 Applications of Phylogenetic Trees

  • Understanding evolutionary relationships
  • Identifying transmission clusters
  • Investigating population structure
  • Tracking pathogen spread

1.3 Newick Format

Phylogenetic trees are commonly stored in Newick format. This format uses parentheses to represent branching structure. 

(A:0.1,B:0.2,(C:0.3,D:0.4):0.5);

1.4 Methods for Inferring Trees

Tree inference methods include:

  • Distance-based methods
  • Maximum Parsimony
  • Maximum Likelihood
  • Bayesian inference

1.5 Assessing Tree Uncertainty (Bootstrap)

Bootstrap analysis evaluates confidence in tree branches by resampling alignment columns and reconstructing trees repeatedly.

2. Multiple Sequence Alignments

Phylogenetic inference requires a multiple sequence alignment (MSA). The alignment ensures homologous positions are compared across sequences.

Poor alignment quality can significantly affect tree accuracy. Always inspect alignments before proceeding.

3. Building a Phylogenetic Tree

3.1 Extracting Variable Sites with SNP-sites

SNP-sites extracts variable positions from an alignment to produce a reduced alignment containing only informative sites. 

snp-sites -o core_snps.aln alignment.fasta

3.2 Tree Inference with IQ-TREE

IQ-TREE is a widely used maximum likelihood phylogenetic inference tool. 

iqtree2 -s core_snps.aln -m GTR+G -bb 1000

3.3 Rooting a Phylogenetic Tree

Trees can be rooted using an outgroup or midpoint rooting. Proper rooting is essential for interpreting evolutionary direction.

4. Summary

  • Phylogenetic trees describe evolutionary relationships.
  • High-quality alignments are critical.
  • Maximum likelihood methods are widely used in bacterial genomics.
  • Bootstrap values provide branch support.
