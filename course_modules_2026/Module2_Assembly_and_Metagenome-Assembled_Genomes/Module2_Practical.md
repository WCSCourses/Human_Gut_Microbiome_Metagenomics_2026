# Assembly and Metagenome-Assembled Genomes (MAGs)
---

# Exercise overview

In this practical session, we will reconstruct microbial genomes from metagenomic sequencing data using the **metaWRAP pipeline**.

Students will run the workflow on **one metagenomic sample** in order to:

1. Assemble sequencing reads into **contigs**
2. Group contigs into **genome bins**
3. Refine bins into **high-quality MAGs**
4. Assess genome quality using **CheckM**

This exercise demonstrates the core workflow used in **genome-resolved metagenomics**.
