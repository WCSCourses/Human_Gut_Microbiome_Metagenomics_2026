# Introduction

Shotgun metagenomic sequencing generates millions of short DNA fragments originating from many different organisms within a microbial community. These fragments are typically between **100–300 base pairs long**, whereas microbial genomes are typically **1–10 million base pairs**.

To reconstruct genomes from these fragments, computational methods are used to assemble overlapping reads into longer sequences. This process is called **metagenomic assembly**.

When assembled sequences from the same organism are grouped together, the result is a **metagenome-assembled genome (MAG)**.

MAGs allow researchers to:

- Recover genomes of **uncultured microbes**
- Identify **novel microbial species**
- Study **metabolic pathways**
- Explore **microbial ecology**
- Investigate **strain variation**

Genome-resolved metagenomics has dramatically expanded our understanding of microbial diversity.

# Key Concepts

## Reads

Reads are the **short DNA sequences generated by sequencing machines**. Reads are typically **paired-end**, meaning each DNA fragment is sequenced from both ends.

Example:
ATCGTACGTA
CGTACGTAGC
TACGTAGCTA

## Contigs
A **contig** (contiguous sequence) is a longer sequence reconstructed by overlapping reads.

Example: ATCGTACGTAGCTA
Contigs represent **assembled fragments of genomes**.

---

## Scaffolds
A **scaffold** is a set of ordered contigs connected using paired-end information.

Example:
Contig1 — gap — Contig2 — gap — Contig3
Scaffolds provide a **higher-level genome structure**, but gaps may remain.

---
# Why Assembly Matters in Metagenomics

Read-based profiling tools (e.g., MetaPhlAn, Kraken2) identify **taxonomic composition**, but they do not reconstruct genomes.
A
ssembly enables:
1. Genome reconstruction: recover near-complete genomes directly from environmental samples.
2. Discovery of novel microbes: many microbial species cannot be cultured.
3. Functional interpretation: link genes and pathways to specific organisms.
4. Evolutionary analysis: compare genomes between strains and species.
5. Reference genome generation: MAGs can become reference genomes for future studies.

# Step 1: Metagenomic Assembly
metaWRAP uses **MEGAHIT** for metagenomic assembly.

MEGAHIT is designed for:

- large datasets
- low memory usage
- complex microbial communities


```bash
metawrap assembly \
    -1 sample_R1.fastq.gz \
    -2 sample_R2.fastq.gz \
    -o assembly_output \
    -t 16 \
    -m 64
 ```
-1 is forward reads
-2 is reverse reads
-o is the output directory
-t is the number of threads
-m is the memory in GB allocated for this specific run

The main output file:
```text
assembly_output/final_assembly.fasta
```
